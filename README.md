### QOD-SimVis: a Visual Similarity Measure between Qualitative Object Descriptors

Dataset QOD-SimVis composed by 15 scenes related to the Cube Comparison Test.

Prolog code for SWI-Prolog. For non-experts, you can execute the code in the online platform: https://swish.swi-prolog.org

Authors: Zoe Falomir (zfalomir@cs.umu.se) and Vicent Costa (vicent@iiia.csic.es)
